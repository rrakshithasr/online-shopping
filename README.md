# 🛒 Online Shopping Orders Analysis using Pandas

## 📌 Project Overview
This project demonstrates basic data analysis using the **Pandas** library in Python. It analyzes an online shopping dataset by calculating revenue, filtering orders, sorting data, and generating useful business insights.

## 🎯 Objectives
- Learn the basics of Pandas.
- Read and write CSV files.
- Perform data analysis on shopping orders.
- Practice filtering, sorting, and summarizing data.

## 🛠️ Technologies Used
- Python 3
- Pandas
- Jupyter Notebook

## 📂 Dataset
The dataset contains the following columns:
- Order_ID
- Customer
- Product
- Quantity
- Price
- Status

## ✨ Features
- Create a sample shopping dataset.
- Save and read CSV files.
- Display dataset information.
- Generate summary statistics.
- Calculate total revenue.
- Find the highest and lowest order value.
- Filter delivered and pending orders.
- Sort orders by total amount.
- Count orders by status.
- Save the updated dataset.

## 📚 Pandas Concepts Covered
- DataFrame
- read_csv()
- to_csv()
- head()
- tail()
- info()
- describe()
- shape
- columns
- dtypes
- Boolean Filtering
- sort_values()
- value_counts()
- sum()
- mean()
- idxmax()
- idxmin()

## ▶️ How to Run
1. Open Jupyter Notebook.
2. Create a new notebook.
3. Copy and paste the project code into a single cell.
4. Run the cell.
5. The program will create `orders.csv` and `updated_orders.csv`.

## 📈 Output
The program displays:
- Complete dataset
- Dataset information
- Summary statistics
- Total revenue
- Highest and lowest order
- Delivered and pending orders
- Sorted orders
- Order status count
